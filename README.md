# Lumisense

Аудиоплеер под Windows 11 в стиле Fluent Design: Mica-фон, скруглённые элементы, кастомный
заголовок окна, тёмная/светлая тема. Три вида окна (прямоугольное, квадратное и компактный
мини-плеер), плейлист по группам/папкам с избранным, обложки, теги, история изменений с
автоматическим номером версии — и обновление одним кликом.

## Возможности

- **Воспроизведение** mp3/wav/wma/flac/m4a/aac/ogg через NAudio: play/pause/stop, вперёд/назад,
  перемотка мышью по прогресс-бару, регулировка громкости (в том числе плавная логарифмическая
  у самого нуля), автопереход к следующему треку
- **Плейлист по группам**: добавление отдельных файлов, целых папок (с подпапками) или пустой
  папки для ручного наполнения; каждую группу можно включать/выключать в проигрывании отдельно,
  сворачивать/разворачивать список треков, проверять на новые файлы, появившиеся на диске уже
  после добавления
- **Избранное** — сердечко на любом треке добавляет его в отдельный виртуальный плейлист
  «Избранное», с переключением между ним и обычным плейлистом прямо из шапки
- **Шафл и повтор** (без повтора / по кругу / повтор одного трека)
- **Три вида окна**: прямоугольный, квадратный (крупная обложка) и мини-плеер — компактное
  окошко поверх других окон, которое можно закрепить, таскать с прилипанием к краям экрана,
  настроить прозрачность; фон мини-плеера подстраивается под цвет текущей обложки
- **Обложки альбомов**: чтение из тегов файла, поиск в интернете, ручная установка своего
  изображения, свойства обложки
- **Редактирование тегов** (название, исполнитель, альбом и т.д.) и свойств трека прямо из
  плеера
- **Системная интеграция Windows 11**: медиа-клавиши клавиатуры (Play/Pause/Next/Prev) и
  настраиваемые пользователем комбинации, значок в трее со своим меню и мини-обложкой, Now
  Playing на панели задач и блокировке экрана через System Media Transport Controls
- **Список изменений** прямо в приложении — с версией, которая считается автоматически по
  смыслу текста изменений (SemVer: добавили что-то по мелочи — patch, крупную новую
  возможность — minor, что-то несовместимое — major), поиском, сортировкой и картинками у
  отдельных записей
- **Восстановление состояния между запусками**: плейлист, последний трек и позиция в нём,
  громкость, шафл/повтор, избранное, вид окна — всё сохраняется и восстанавливается при
  следующем запуске
- **Проверка обновлений** через GitHub Releases — тихая проверка при старте плюс кнопка в
  настройках, установка новой версии в один клик
- Гибкие настройки: сворачивание в трей вместо закрытия, ярлык на рабочем столе, поведение
  мини-плеера и многое другое

## Стек

- **.NET 8** (WPF, `net8.0-windows10.0.19041.0`)
- **[WPF-UI](https://github.com/lepoco/wpfui)** — Fluent-компоненты, `FluentWindow`, Mica-фон,
  системные иконки
- **[NAudio](https://github.com/naudio/NAudio)** — декодирование и воспроизведение аудио
- **[TagLibSharp](https://github.com/mono/taglib-sharp)** — чтение и запись тегов/обложек
- **[SharpVectors](https://github.com/ElinamLLC/SharpVectors)** — рендер SVG-иконок интерфейса
- **Windows Forms** — значок в трее (`NotifyIcon`)

## Как запустить из исходников

1. Установите **Visual Studio 2022** (17.8+) с workload **.NET desktop development**, либо
   просто **.NET 8 SDK**, если хотите собирать из консоли.
2. Склонируйте репозиторий.
3. Из консоли в папке `Lumisense`:
   ```
   dotnet restore
   dotnet run
   ```
   Либо откройте `Lumisense.csproj` напрямую в Visual Studio и нажмите F5.

При первом восстановлении NuGet подтянет все пакеты автоматически.

## Готовая сборка

Собранный установщик (`Lumisense_Setup.exe`) публикуется на странице
[Releases](releases) при каждом новом теге `v*.*.*` — собирается и публикуется
автоматически через `.github/workflows/release.yml` (self-contained `dotnet publish` +
установщик Inno Setup из `Installer/Lumisense.iss`). Сам плеер умеет проверять и подсказывать
про новые версии оттуда же.

## Структура репозитория

```
Lumisense/
├── Installer/Lumisense.iss        — сценарий установщика (Inno Setup)
├── .github/workflows/release.yml  — сборка и публикация релиза по тегу
└── Lumisense/                     — исходники самого плеера
    ├── Lumisense.csproj
    ├── App.xaml / .cs                          — точка входа, подключение тем WPF-UI
    ├── MainWindow.xaml / .cs                   — основное окно: плейлист, воспроизведение
    ├── MiniPlayerWindow.xaml / .cs             — компактный мини-плеер
    ├── AppSettings.cs                          — настройки, сохранение/восстановление состояния
    ├── PlaylistFolder.cs / Favorites.cs        — модель плейлиста и избранного
    ├── CoverArt*.xaml(.cs)                     — окна обложки: поиск, свойства, установка
    ├── Track*.xaml(.cs)                        — свойства трека, редактирование тегов
    ├── SettingsWindow.xaml / .cs                — окно настроек
    ├── Changelog*.cs, ChangelogWindow.xaml(.cs) — список изменений и подсчёт версии по SemVer
    ├── Changelog/changelog.json                — сама история изменений (см. комментарий в
    │                                              ChangelogLoader.cs про формат записей)
    ├── TrayIconManager.cs                       — значок в трее и его меню
    ├── GlobalMediaHotKeys.cs                    — медиа-клавиши клавиатуры и свои хоткеи
    ├── NowPlayingIntegration.cs                 — Now Playing (SMTC) в Windows 11
    ├── UpdateChecker.cs, UpdateAvailableWindow.xaml(.cs) — проверка и установка обновлений
    └── Icons/                                   — SVG-иконки интерфейса и иконка приложения
```
