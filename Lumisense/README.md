# Lumisense (C# + WPF + WPF-UI + NAudio)

Готовый каркас плеера в стиле Fluent Design под Windows 11: Mica-фон, скруглённые
элементы, тёмная тема, кастомный заголовок окна.

## Стек

- **.NET 8** (WPF, `net8.0-windows`)
- **WPF-UI** — Fluent-компоненты, `FluentWindow`, `TitleBar`, `Button`, иконки
- **NAudio** — декодирование и воспроизведение mp3/wav/wma/flac

## Как запустить

1. Установите **Visual Studio 2022** (17.8+) с workload **.NET desktop development**,
   либо просто **.NET 8 SDK**, если хотите собирать из консоли.
2. Распакуйте папку `Lumisense` в удобное место.
3. Из консоли в папке проекта:
   ```
   dotnet restore
   dotnet run
   ```
   Либо откройте `Lumisense.csproj` напрямую в Visual Studio и нажмите F5.

При первом восстановлении NuGet подтянет пакеты `WPF-UI` и `NAudio` автоматически.

## Готовый установщик

Чтобы получить обычный `Setup.exe`, который ставит плеер одним кликом (без Visual
Studio и консоли) — см. `installer/README.md`. Собрать его нужно один раз на компьютере
с Windows (пара бесплатных программ, 10 минут), дальше готовый установщик можно
передавать куда угодно.

## Что уже реализовано

- Открытие одного или нескольких аудиофайлов через диалог
- Плейлист с двойным кликом для переключения трека
- Play / Pause / Stop / Prev / Next
- Перемотка мышью по прогресс-бару
- Регулировка громкости
- Автопереход к следующему треку по окончании текущего
- Отображение текущего/общего времени трека

## Что можно добавить дальше

- Отображение обложки альбома (через `TagLib#` — читает метаданные mp3/flac)
- Эквалайзер (NAudio поддерживает `BiQuadFilter` для полос эквалайзера)
- Визуализация спектра (через `FftEventArgs` в связке с `SampleAggregator`)
- Сохранение плейлиста между запусками (просто сериализовать `_playlist` в JSON)
- Системная интеграция: медиа-клавиши клавиатуры, Now Playing в Windows 11
  (через `SystemMediaTransportControls`)
- Переключение светлой/тёмной темы: `ui:ThemesDictionary` можно менять на лету
  через `Wpf.Ui.Appearance.ApplicationThemeManager.Apply(...)`

## Структура файлов

```

└── MainWindow.xaml.cs      — логика воспроизведения на NAudio
```
