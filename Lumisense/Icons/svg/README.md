# Иконки

Здесь лежат настоящие `.svg`-файлы. Никакой промежуточной конвертации не нужно —
WPF рендерит их напрямую (через библиотеку [SharpVectors](https://github.com/ElinamLLC/SharpVectors),
`SharpVectors.Wpf`).

## Как поменять иконку

1. Замени файл `IconXxx.svg` на другой (экспортированный из Figma, Illustrator,
   Inkscape, взятый из любой библиотеки иконок и т.п.). Имя файла менять не
   нужно — именно по нему иконку находит `SvgPathIcon` в разметке.
2. Пересобери проект. Всё остальное (App.xaml, код кнопок) трогать не нужно.

Чтобы добавить новую иконку — просто положи новый `.svg` в эту папку и
используй его имя (без расширения) как значение `Icon`:

```xml
<local:SvgPathIcon Icon="ИмяФайла" />
```

или из кода:

```csharp
IconResources.Make("ИмяФайла")
```

## Цвет

Иконки монохромные: реальный `fill`/`color` внутри самого `.svg` не важен —
на экране иконка всегда закрашивается цветом `Foreground` того места, где она
показана (в том числе меняется при наведении/нажатии на кнопку, если это
предусмотрено стилем кнопки). Так что можно смело брать любой чёрно-белый SVG
как есть, без правки цветов внутри файла.

## Размер по умолчанию

Необязательный атрибут `data-default-size` на корневом `<svg>` — это размер
иконки в пикселях, который используется, если в месте показа иконки размер не
указан явно (`Size="..."`). Например:

```xml
<svg data-default-size="16" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24">
  <path d="..."/>
</svg>
```

Если атрибута нет — используется размер 20. Это валидный SVG-атрибут (`data-*`),
он ничего не ломает при открытии файла в любом графическом редакторе.

## Список файлов в этой поставке

Большинство иконок — из набора Google Material Symbols (Apache 2.0), 20 штук,
залиты как есть. Ещё 7 файлов (`IconAddFolder`, `IconDismiss`, `IconExpand`,
`IconMinimizeWindow`, `IconMusicNote`, `IconRestoreWindow`, `IconSpeaker`)
восстановлены как настоящие `.svg` из прежних Fluent UI System Icons
(Microsoft, MIT), которые использовались в проекте раньше — чтобы приложение
не потеряло эти иконки при переходе на новую систему.

Четыре иконки — `IconRepeat`, `IconShuffleOn`, `IconSpeaker1`, `IconSpeaker2` —
пока лежат про запас: в текущей разметке они нигде не используются (например,
приложение сейчас переключает только `IconSpeaker` / `IconSpeakerMute`, без
промежуточных уровней громкости), но уже готовы, если решишь их подключить —
просто впиши `Icon="IconSpeaker1"` и т.п. в нужном месте.
